opml.wiki

An experiment in rendering Federated Wiki's as OPML files.

Each page is linked via an OPML include.

Examples

Main OPML file for a federated wiki:

http://opml.wiki/fed.wiki.org.opml

OPML file for a page:

http://opml.wiki/fed.wiki.org/about-federated-wiki.opml

JSON also works:

http://opml.wiki/fed.wiki.org.json

http://opml.wiki/fed.wiki.org/about-federated-wiki.json

Try Other Wiki

This renders any site that conforms to the fedwiki json files.

Browsing

This works great in an outliner like Little Outliner.

Simply sign on with Twitter, then select "Subscribe to outline…" from the File menu.

Paste in the URL to an opml file like "http://opml.wiki/fed.wiki.org.opml" and see how it looks.
